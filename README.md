# ChangeReady360

Wilkommen im neuen Entwicklungsbranch von CHangeReady360. MIt diesem Branch beginnt offiziell die Implementierungsphase des Projekts. Bisher befand sich ChangeReady in der Konzeptions-Planungsphase. Dieser Branch dient als Grundlage für den Aufbau der gesamten Codebasis. 


Dieses Projekt besteht aus:

Backend: Spring Boot (Java 17, Gradle)

Frontend: Angular (Node 22, Nginx)



Deployment: Docker & docker-compose

Projekt starten:


docker-compose up --build

Zugriff:

Frontend → http://localhost:4200

Backend → http://localhost:8080

Projekt im Hintergrund starten:

docker-compose up --build -d

Laufende Container prüfen:

docker ps

Projekt stoppen:

docker-compose down
